## For Your Browser

**Recommended setup:**
1. Open all 4 URLs in separate tabs
2. Pin the tabs in your browser
3. During demo:
   - Tab 1: Aspire Dashboard (main view)
   - Tab 2: Payments API Swagger (send requests)
   - Tab 3: Core Bank API Swagger (check inbox)
   - Tab 4: Jaeger (show traces)

**Pro tip:** In Aspire Dashboard, click on any service's "http" endpoint, then manually add `/swagger` to the URL.
